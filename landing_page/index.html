Camera Feed

View live camera feed to monitor surroundings visually.

About the Project

This project was inspired by the need for advanced environmental monitoring solutions. The Environmental Data Monitoring Robot aims to provide comprehensive real-time data on various environmental parameters to help in better decision-making and environmental protection.

This is a Portfolio Project for Holberton School.
